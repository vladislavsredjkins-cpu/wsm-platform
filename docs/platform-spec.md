# World Strongman Platform

## Overview
World Strongman Platform is a global infrastructure for managing strongman competitions, rankings, and athlete profiles.

The platform supports:

- Event lifecycle management
- Live judging
- Multiple competition formats
- Global ranking calculation
- Athlete registry
- Official protocol generation

## Competition Formats

### CLASSIC
Standard strongman competition with multiple disciplines.

### PARA
Adaptive strongman format.

### RELAY
Two athletes performing sequential disciplines.

### TEAM_BATTLE
Two-athlete teams competing across disciplines.
